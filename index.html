Hotel Reservation Cancellation Prediction with In-Depth Analysis

In this project, the goal was to predict hotel reservation cancellations using logistic regression. The process involved conducting exploratory data analysis (EDA) to identify key features and relationships, cleaning and pre-processing the data, and applying a logistic regression model with hyperparameter tuning using Grid Search CV. The performance of the model was evaluated using cross-validation and a confusion matrix. Python, Scikit-learn, Pandas, NumPy, Matplotlib, and Seaborn were utilized for the analysis. The final model was able to accurately predict hotel reservation cancellations with 98% accuracy.

The analysis of the swine flu outbreak of 2009 suggests that population density played a significant role in the spread of the disease in the United States and Mexico.
